# Placemark

Node-based webapp that allows users to create and manage points of interests using an interactive map or raw data.

## Description

Users can create points of interest using the interactive map or by manually adding the latitude and longitude numbers. 
A poi holds summary, name and description. Poi's can be organised/viewed by category.

## Getting Started

### Installing

* Run `npm install` at project root level.
* Edit env_example to env and adapt it to your use.

### Executing program

* Run `npm run start`
* Access `localhost:3000` on any web browser
